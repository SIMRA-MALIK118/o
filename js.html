🔹 What is JavaScript?

JavaScript is a programming language that adds interactivity, controls multimedia, animates images, and much more on webpages.

🔹 Basic Syntax

console.log('Hello, world!');

🔹 Variables

let name = 'Ali';
const pi = 3.14;

🔹 Functions

function greet() {
  alert('Welcome to JavaScript!');
}

💡 Pro Tip

JavaScript runs in the browser. Use it to make your websites interactive and responsive.
