🎨 CSS Basics

Style your webpages with colors, layouts, and more

🔹 What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and how elements are positioned.

🔹 Basic CSS Syntax

CSS rules consist of selectors and declarations:

selector {
  property: value;
}

🔹 Example CSS Rule

p {
  color: blue;
  font-size: 18px;
}

💡 Pro Tip

Use CSS to separate content (HTML) from design (CSS) for cleaner code and easier maintenance.
